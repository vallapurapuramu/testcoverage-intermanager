# testcoverage-intermanager

```
 1)The main thing to be installed is @vue/cli-plugin-unit-jest 
      We can install that with the command npm i @vue/cli-plugin-unit-jest
      we must install package in the project directory
2)Once we install we can add following command to package.json
  "test:unit": "vue-cli-service test:unit --watch"
  
3)npm run test:unit it will the test report in coverage folder
```
